# Hackathon
*AWS_Hackathon_of_AI_Idol*
  
- 目標：打造陪伴粉絲、協助偶像業務的虛擬分身  
- 核心功能：智慧聊天互動、虛擬代言或廣告  
- 增強陪伴感：生日、婚慶等粉絲重要日子也能即時互動祝賀  
- 虛擬偶像陪玩：在遊戲機(如switch)上和虛擬偶像互動一起玩遊戲  
- 虛擬偶像老師：讓AI偶像擔任老師，更能夠激起學習動力
  
## AWS工具架構  
1. 智慧聊天語音系統  
  - EC2  
  - Amazon S3  
  - Amazon Bedrock  
  - CloudFront  
  - API Gateway  
2. 虛擬人  
  - Musetalk:利用提供的企業數據和生成的語音進行合成，建立虛擬偶像。
